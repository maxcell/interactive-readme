CSS Selectors

We've learned that HTML is for content and CSS is for style.

But which HTML elements we are going to style?

We need a way to select elements. In CSS, we select elements with selectors.

Let's see some examples of selectors in action.

Some selectors
body
div
button.primary
#submit
h1, h2, h3, h4

Selectors select HTML elements. Which elements are selected depends on the selector. Let's start with the basic rules.

Selectors like body or div select all the elements of that type.
